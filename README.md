# Discretisation invariance

A demonstration of discretisation invariance.

Datasets:
1. Viscous Burgers, [code](https://github.com/4gnskq5g2s-collab/discretisation_invariance/blob/main/datasets/Burgers.py), [download](https://disk.yandex.ru/d/9-33PCZX-w2Dyg)
2. Stationary diffusion, [code](https://github.com/4gnskq5g2s-collab/discretisation_invariance/blob/main/datasets/Diffusion.py), [download](https://disk.yandex.ru/d/bb61QJCzEK9fnQ)

Architectures:
1. DeepONet, [paper](https://arxiv.org/abs/1910.03193), [code](https://github.com/4gnskq5g2s-collab/discretisation_invariance/blob/main/architectures/DeepONet.py)
2. FNO, [paper](https://arxiv.org/abs/2010.08895), [code](https://github.com/4gnskq5g2s-collab/discretisation_invariance/blob/main/architectures/FNO.py)
3. U-Net, [paper](https://arxiv.org/abs/1505.04597), [code](https://github.com/4gnskq5g2s-collab/discretisation_invariance/blob/main/architectures/UNet.py)

Main dependencies:
1. [jax](https://github.com/jax-ml/jax)
2. [optax](https://github.com/google-deepmind/optax)
3. [equinox](https://github.com/patrick-kidger/equinox)
